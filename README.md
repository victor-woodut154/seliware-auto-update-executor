## About Seliware

Seliware is a free, modern Roblox script executor made for Windows. It gives you a direct way to load and run Lua scripts inside Roblox games, whether you are automating repetitive actions, extending gameplay, or using functions that are not available in the standard client. The workflow stays simple: open the executor, inject into Roblox, then pick a script from the included hub.

It also keeps a persistent queue stored in SQLite so your execution history remains available after closing the app. An integrated auto-update system helps keep compatibility aligned with Roblox patches, while the multi-language interface makes the tool easier to use in different regions. There are no activation keys or extra charges - everything is available from the start.

## FAQ

**Is Seliware safe to use?**  
Like any external tool that works with Roblox, you use it at your own discretion. It does not include intentionally harmful code, but we cannot promise that it will never trigger detections or security warnings.

**Will it work after a Roblox update?**  
Yes. The auto-update engine delivers compatibility fixes shortly after Roblox patches are released. For best results, always run the latest build.

**How does Seliware compare to other free executors?**  
Seliware stays completely keyless and adds a persistent script queue, a large built-in hub, and multi-language support, which are often reserved for paid tools.

**Can my Roblox account be banned?**  
Script executors are against Roblox's Terms of Service, so bans are possible. We recommend using an alternate account and proceeding carefully.

**Where are my scripts stored?**  
Executed scripts are stored locally in an SQLite database under the `Data` folder. The bundled script hub also lives on your device.
